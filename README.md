# Stud OS

A minimalistic, 1-bit retro developer toolkit and gamified study environment built with vanilla HTML, CSS, and JavaScript. It is designed to act as a focused operating system to help you log deep work hours, ship projects, and test your web development knowledge.

## Features

* **1-Bit Aesthetic:** High-contrast black and white UI with CSS CRT scanlines.
* **Audio Player:** Built-in local audio player with a canvas frequency visualizer.
* **Syntax Typer:** A WPM-tracking typing test using real-world code snippets.
* **Black Market:** An auto-saving local store to unlock features using earned XP.
* **Introvert Mode:** A dark mode toggle for low-strain coding sessions.

## Minigames

* **The 50-Q Gauntlet:** Rapid-fire HTML, CSS, and JS trivia.
* **CSS Sniper:** Target specific CSS selectors in a dynamic arena.
* **Tag Match Memory:** A concentration game using HTML tags.
* **Flex Master:** A spatial puzzle for mastering CSS Flexbox properties.
* **Mega Arcade:** A 10-in-1 trivia engine covering Git, HTTP, Vim, binary, and more.
