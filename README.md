Stud OS is a comprehensive, 1-bit retro developer toolkit and gamified study 
environment. Built entirely with Vanilla JavaScript, HTML, and CSS, this 
system acts as a functional operating system designed to lock in your focus, 
test your dev knowledge, and help you farm Pages while you ship projects.

+-----------------------------------------------------------------------------+
| ::: [ S Y S T E M   F E A T U R E S ] :::                                   |
+-----------------------------------------------------------------------------+

[+] 1-BIT CRT AESTHETIC
    High-contrast, checkerboard-dithered UI complete with CSS-rendered CRT 
    scanlines and custom block scrollbars.

[+] GOLDEN RATIO ARCHITECTURE
    All spatial sizing, padding, and animations are meticulously scaled using 
    the Golden Ratio (1.618) for subconscious, organic visual harmony.

[+] DEEP WORK AUDIO ENGINE
    Integrated local audio player featuring a dynamic <canvas> frequency 
    visualizer and precision playback scrubber.

[+] SYNTAX TYPER
    A strict, WPM-tracking typing test featuring real-world code snippets 
    (FastAPI, React, Framer Motion) to warm up your mechanical switches.

[+] THE BLACK MARKET
    An auto-saving local storage shop where you spend your hard-earned XP to 
    unlock advanced tools, protocols, and micro-games.

[+] INTROVERT MODE
    A built-in theme toggle that inverts the stark 1-bit colors for 
    late-night, low-strain, isolated coding sessions.

+-----------------------------------------------------------------------------+
| ::: [ A R C A D E   P R O T O C O L S ] :::                                 |
+-----------------------------------------------------------------------------+

Earn XP by proving your development knowledge through several integrated engines:

> THE 50-Q GAUNTLET
  A rapid-fire trivia engine testing deep HTML, CSS, and JS mechanics.

> CSS SNIPER
  Practice your DOM targeting by shooting specific CSS selectors in a chaotic 
  checkerboard arena.

> TAG MATCH MEMORY
  A classic concentration game utilizing escaped HTML tags.

> FLEX MASTER
  A spatial reasoning puzzle to master `justify-content` and `align-items`.

> MEGA ARCADE
  A randomized 10-in-1 trivia engine covering Git commands, HTTP status codes, 
  Vim motions, binary conversion, and Python/JS bug hunting.

+-----------------------------------------------------------------------------+
| ::: [ B O O T   S E Q U E N C E ] :::                                       |
+-----------------------------------------------------------------------------+

1. CLONE THE DIRECTORY
   $ git clone https://github.com/kaustubh-cell/stud-os.git

2. NAVIGATE TO REPOSITORY
   $ cd stud-os

3. INITIALIZE LOCAL SERVER
   Because the Web Audio API requires a secure context to analyze files, you 
   must serve the files using a local web server to avoid CORS blocks.
   
   * Python : $ python -m http.server 8000
   * Node   : $ npx serve
   * VS Code: Execute "Live Server" extension

4. EXECUTE
   Open your browser and navigate to http://localhost:8000
