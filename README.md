#  OS.K8BH - DevUtility MA

A comprehensive, 1-bit retro developer toolkit and gamified study environment. Built entirely with Vanilla JavaScript, HTML, and CSS, this project acts as a functional operating system designed to keep you focused, testing your dev knowledge while you ship projects. 

Whether you are grinding towards a 170-hour hackathon goal or farming Pages in the Hack Club YSWS ecosystem, OS.K8BH provides the ultimate high-focus environment.

##  Features

* **1-Bit CRT Aesthetic:** High-contrast, checkerboard-dithered UI complete with CSS-rendered CRT scanlines and custom chunky scrollbars.
* **Golden Ratio Architecture:** All spatial sizing, padding, and animations are meticulously scaled using the Golden Ratio (`1.618`) for subconscious, organic visual harmony.
* **Deep Work Audio Player:** Integrated local audio engine featuring a dynamic `<canvas>` frequency visualizer and playback scrubber.
* **Syntax Typer:** A strict, WPM-tracking typing test featuring real-world code snippets (FastAPI, React, Framer Motion) to warm up your fingers.
* **The Black Market:** An auto-saving local storage shop where you spend your hard-earned XP to unlock advanced tools and protocols.
* **Introvert Mode:** A built-in theme toggle that inverts the stark 1-bit colors for late-night, low-strain coding sessions.

##  Micro-Game Suite (The Arcade)

Earn XP by proving your development knowledge through several integrated minigames:
* **The 50-Q Gauntlet:** A rapid-fire trivia engine testing deep HTML, CSS, and JS mechanics.
* **CSS Sniper:** Practice your DOM targeting by shooting specific CSS selectors in a chaotic arena.
* **Tag Match Memory:** A classic concentration game utilizing escaped HTML tags.
* **Flex Master:** A spatial reasoning puzzle to master `justify-content` and `align-items`.
* **Mega Arcade:** A randomized 10-in-1 trivia engine covering Git commands, HTTP status codes, Vim motions, binary conversion, and Python/JS bug hunting.

##  Tech Stack

* **Frontend:** Pure HTML5 & CSS3 (Zero external libraries)
* **Logic:** Vanilla JavaScript (ES6+)
* **State Management:** Web Storage API (`localStorage`)
* **Audio:** Web Audio API (`AudioContext`, `AnalyserNode`)
